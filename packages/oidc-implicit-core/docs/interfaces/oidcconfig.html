Optional qr

qr: { channel_qr: string; channel_redirect: string; web_socket: string }
  • Defined in models/config.models.ts:97

Config object for QR login with websocket

Type declaration

  • channel_qr: string

    Websocket channel for QR code rendering. When subscribing to this channel you will receive a code to parse to QR and render on screen. So this sets up your session to login with QR.

  • channel_redirect: string

    Websocket channel for the Redirect. When subscribing to this channel you will receive a 302 redirect once the QR was scanned in the Vodafone App. So this will keep listening if your session started with the QR channel was succesfully scanned somewhere.

  • web_socket: string

    Websocket BASE URL. Connect to this URL to have access to the subscription channels
